How to Disable IPv6 Permanenetly in Linux

In this blog we will see how to make persistent changes to IPv6 protocol in your Linux computer
Published July 17, 2024 | 1 min

Disable IPv6

Disable IPv6 Temporarily

For all Network Interfaces
sudo sysctl -w net.ipv6.conf.all.disable_ipv6=1
sudo sysctl -w net.ipv6.conf.default.disable_ipv6=1
For a Specific Network Interface
sudo sysctl -w net.ipv6.conf.{eth0}.disable_ipv6=1

Replace {eth0} with teh interface name for which you want to disable the IPv6

Disable IPv6 Permanenetly

Modify the /etc/sysctl.conf file to disable IPv6 permanenetly

sudo nano /etc/sysctl.conf

Then modify the entries as follow

For All Network Interfaces
net.ipv6.conf.all.disable_ipv6=1
net.ipv6.conf.default.disable_ipv6=1
For a Specific Network Interface
net.ipv6.conf.{eth0}.disable_ipv6=1

Now restart the computer to apply the changes

sudo reboot

Enable IPv6

Enable IPv6 Temporarily

For all Network Interfaces
sudo sysctl -w net.ipv6.conf.all.disable_ipv6=0
sudo sysctl -w net.ipv6.conf.default.disable_ipv6=0
For a Specific Network Interface
sudo sysctl -w net.ipv6.conf.{eth0}.disable_ipv6=0

Enable IPv6 Permanenetly

For All Network Interfaces
net.ipv6.conf.all.disable_ipv6=0
net.ipv6.conf.default.disable_ipv6=0
For a Specific Network Interface
net.ipv6.conf.{eth0}.disable_ipv6=0

Now restart the computer to apply the changes

sudo reboot
