How to Formart a USB Stick From Terminal

Published June 30, 2024

At first you need to identify the drive name, you can do so as follows

Deleting the partitions

In the terminal type in

df

which shows the list of Drives and it’s partitions. Choose the correct drive, in my case it is the /dev/sda drive. Now we would use the fdisk terminal utility for modifications in the drive, and type in

sudo fdisk /dev/sda

which lets you in a interactive session, where you can perform certain operations on the selected drive, like addition, deletion, renaming of a partition. In our case we will delete all the existing partitions, so type in

d

as it suggests d is for deletion operation of a selected partition, and by default it selects the first partition (in case of multiple existing partitions or the only partition). We will delete the partition. But only after you perform the write operation, the process actually starts. So, we type in

w

and the partition is deleted. And now our USB-stick is ready to be formatted.

Formatting the drive

First we have to unmount the drive, by

sudo umount /dev/sda

to format in FAT32 format, type in

sudo mkfs.vfat /dev/sda

And it is done!

You can unplug and replug the drive, and you can see the formatted the pendrive in your file manager.
